Work

For more specific information regarding accomplishments for work, please refer to the Resume section. This section is meant to provide a more personal perspective on the jobs I've had.

Amazon (Summer 2020)

For Summer 2020, I interned at Amazon in the Customers Acquisition team. This team was focused on acquiring and verifying potential Amazon Business customers. My intern project was to create a tool that would help developers debug any issues that come up with a customer's registration and verification process by showing an ordered Step Function workflow execution history, and multiple smaller features. This tool was a full-stack application which I created using various AWS technologies through CloudFormation such as AWS Lambda, CloudWatchEvents, EventBridge, SecurityTokenService, IAM Roles, SQS Queues, Dead-Letter Queues, and Step Functions.

Overall, it was a successful internship where I learned a lot and gained valuable experience. I was able to deploy my tool to production and it is now currently being used by developers throughout Amazon Business. I've greatly improved on being able to thoroughly design a project from scratch, collaborating with others, and writing efficient code.

UCSC Engineering Lab (Fall 2019 - Winter 2020)

During the school year, I spent some time helping out Professor David Lee's lab. Here, I helped develop the lab's main page and a web application that teaches others how to code, Causeway, using Angular. This experience was valuable because I had the chance to develop my skills in web development while in school.

Personal Capital (Summer 2019)

For Summer 2019, I interned at Personal Capital, an online financial advisor and personal wealth management company. I was on the Mobile development team, specifically the QA team. Here, I developed my skills in mobile development and automated UI testing through Android Espresso. This was a very fun and new experience for me as I had never focused on testing so heavily. I believe that my ability to test efficiently has been due to my time here.
